Decisiones Técnicas

Modelo Principal

es_core_news_sm

Usado para tokenización, lemas, POS y stopwords en español mediante spaCy.

Arquitectura

Código Lineal

Se mantuvo corto y directo para facilitar la sustentación oral del proyecto.

Manejo de Errores

Instalación Asistida

Si spaCy no está instalado, el script indica el comando exacto necesario.

Correcciones Aplicadas

El pipeline perfecciona el análisis de texto implementando las siguientes correcciones gramaticales y semánticas:

Tiempos Verbales

Basado en el estándar EAGLES.

Lematización

Normalización precisa de adjetivos.

Artículos Indefinidos

Clasificación y detección correcta.

Palabras Ambiguas

Resolución semántica según contexto.

Locuciones

Identificación de expresiones compuestas.

Modos de Ejecución

bash - configuración
# 1. Entorno Virtual
$ python -m venv .venv
$ .venv\Scripts\Activate.ps1
$ pip install -r requirements.txt
$ python -m spacy download es_core_news_lg

# 2. Despliegue Cloud
$ cat Procfile
web: gunicorn app:app

Ejecución Web (Recomendada)

Inicia el servidor Flask y la interfaz gráfica completa.

python app.py → Abre en navegador: http://localhost:5000

Ejecución CLI (Terminal)

Para pruebas rápidas y procesamiento batch desde consola.

python pipeline_nlp.py python pipeline_nlp.py --text "Tu texto aquí"
